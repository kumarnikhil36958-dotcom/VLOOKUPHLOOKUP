VLOOKUP & HLOOKUP

Learn how to search and retrieve data from spreadsheets — interactively, practically, and visually.

What is VLOOKUP?

VLOOKUP = Vertical Lookup. It searches down a column to find a value, then returns something from the same row in another column. Think of it like looking up a student's name in a register and returning their marks.

=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])
lookup_valueThe value you want to find (e.g., a student ID or name)
table_arrayThe range of cells that contains your data (the whole table)
col_index_numWhich column number to return data from (1 = first column)
range_lookupFALSE = exact match (recommended), TRUE = approximate match

Real-Life Example

Imagine you have 500 students. You type a Student ID in cell H2, and want their Math marks automatically in H3. You would write:

=VLOOKUP(H2, A2:E501, 3, FALSE)

✅ H2 = the ID you type  |  A2:E501 = your data table  |  3 = Math is column 3  |  FALSE = exact match

What is HLOOKUP?

HLOOKUP = Horizontal Lookup. It searches across a row to find a value, then returns something from the same column in a row below. Perfect when your data headers run left to right (horizontally).

=HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])
lookup_valueThe value to find in the first row of the table
table_arrayThe range of cells that contains your horizontal data
row_index_numWhich row number to return data from (1 = header row)
range_lookupFALSE = exact match (recommended), TRUE = approximate match

💡 Quick Tips
TIP 1Always use FALSE for range_lookup in most cases. TRUE can give wrong results if data isn't sorted.
TIP 2Lock your table range with $ signs (e.g., $A$2:$E$100) so it doesn't shift when you copy the formula.
TIP 3VLOOKUP can only look right. If your return column is to the left of your search column, use INDEX-MATCH instead.
TIP 4If you get #N/A, the value wasn't found. Check for extra spaces or spelling mismatches.
